# RdpCheck
Cmdline RDP Login PassWord Check Tool (Default Port 3389)

### Usage
```Bash
Usage
RdpCheck Server User Pass
RdpCheck Server:Port User Pass
RdpCheck Server User Pass Domain
RdpCheck Server:Port User Pass Domain
```
